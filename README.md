# Airline-Data-Analysis
## Project Overview
This project focuses on analyzing airline operations data to extract meaningful insights related to performance, profitability, passenger trends, and delays. 
The complete workflow includes data cleaning, analysis, visualization, and dashboard creation using multiple tools and technologies.
The objective of this project is to demonstrate end-to-end data analysis skills by transforming raw airline data into actionable insights and interactive dashboards.

## Key Objective
Clean and preprocess raw airline data
Analyze operational performance (flights, delays, passengers)
Evaluate profit trends and margins
Build interactive dashboards for decision-making
Perform SQL-based analysis for deeper insights

## Tools
Python (Pandas, NumPy) → Data cleaning & preprocessing
SQL (MySQL) → Data querying and analysis
Power BI → Interactive dashboard creation
Microsoft Excel → Data visualization & dashboard
Matplotlib / Seaborn → Data visualization

## Project Workflow
1. Data Cleaning & Preprocessing
Handled missing/null values using Pandas
Removed inconsistencies and formatted data
Prepared dataset for analysis

2. Exploratory Data Analysis (EDA)
Analyzed flight trends by month
Studied passenger distribution
Evaluated delay patterns
Identified profit trends

4. SQL Analysis
Wrote queries to extract insights such as:
Total flights and passengers
Profit distribution
Delay analysis
Monthly performance trends

6. Data Visualization
Created visualizations using Python libraries
Built charts for trend analysis

8. Dashboard Development
 Power BI Dashboard
Interactive and dynamic dashboard
KPIs: Total Profit, Passengers, Delay, Profit Margin
Filters for deeper insights

 Excel Dashboard
